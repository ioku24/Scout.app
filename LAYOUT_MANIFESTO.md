# SCOUT LAYOUT MANIFESTO (V4.0 - PREMIUM SAAS)

## 1. Visual Foundation
- **Backdrop**: Pure white (`#FFFFFF`) with a subtle noise/texture (optional) or solid `#FDFDFD`.
- **Surfaces**: White cards with extremely soft, multi-layered shadows (e.g., `shadow-[0_8px_30px_rgb(0,0,0,0.04)]`).
- **Borders**: 1px `slate-100` for definition, `slate-200` for active states.
- **Primary Accent**: Electric Blue (`#2563eb`).

## 2. The Extraction Console (Redesign)
- **Constraint**: Input fields must be proportional. No excessively wide single-line inputs.
- **Grouping**: Geography and Radius must be treated as a "Coordinate Pair" next to the primary CTA.
- **Interactive States**: Textareas should glow with a soft blue outer ring when focused. 
- **Dictation**: The microphone icon is a floating utility inside the textarea, expanding into a waveform signal only when active.

## 3. The Lead Dossier (Compact 3-Col Grid)
- **Grid Strategy**: 3 columns on Desktop, 1 on Mobile.
- **Asset Triage**: 
  - **Social Nodes**: 24px circular icons with brand-colored hover states.
  - **Website**: A clean external link icon next to the domain name.
- **Content Hygiene**: 
  - Title: Oswald Bold (Uppercase).
  - Subtitle: Address (Muted Slate).
  - Match Logic: A dedicated light-slate container with a high-contrast label.

## 4. Shadows & Depth
- **Level 0**: Background.
- **Level 1**: Standard UI Cards (Subtle shadow).
- **Level 2**: Hovered Lead Cards (Deep soft shadow with a hint of blue tint).
- **Level 3**: Modals and Pop-overs.
